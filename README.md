## Zombie

### Installation
Install the gem:

    gem install sinatra

And run with:

    ruby zombie.rb

or

    rackup -p 4567

View at: http://localhost:4567


### Technologies Used
 - `Sinatra` - A Web Application Server written in ruby.
 - `Pony` - For sending mails
